# SymbolBox · 여기있어 특수문자

> 복사 한 번으로 끝내는 특수문자 모음집

바이브코딩으로 만든 단일 HTML 파일 특수문자 모음 웹앱입니다.  
설치 없이 브라우저에서 바로 열어 쓸 수 있고, 클릭 한 번으로 클립보드에 복사됩니다.

---

## 주요 기능

- **클릭 복사** — 문자 카드를 누르면 즉시 클립보드에 복사, 토스트 메시지로 확인
- **최근 복사** — 최대 12개의 최근 복사 문자를 상단에 표시 (localStorage 저장)
- **이름 검색** — 문자 이름(한글)으로 실시간 검색, 검색 시 자동으로 전체 탭 전환
- **탭 필터** — 13개 카테고리 탭으로 원하는 분류만 빠르게 탐색
- **리플 애니메이션** — 카드 클릭 시 인터랙션 피드백

| 별·도형 | 별, 사각형, 원, 삼각형, 하트, 꽃 등 |
| 화살표 | 단방향·양방향·대각선 화살표 |

## 문서작성 탭 상세

대한민국 행정업무 운영 규정 기준 공문서 특수문자를 7개 블록으로 구성합니다.

### 항목 구분 기호 (규정 순서)

공문서 항목 표시 순서를 시각적 흐름으로 안내합니다.

```
1.  →  가.  →  1)  →  가)  →  (1)  →  (가)  →  ①
첫째   둘째   셋째   넷째   다섯째  여섯째   일곱째
```

원숫자 ① ~ ⑳ 포함.

### 보조 항목 기호

규정에 명시된 보조 기호: `□` `○` `-` `·` 및 도형 계열

### 괄호 및 인용 부호

법령·제목 인용에 쓰이는 `「」` `『』` `〈〉` `《》` `〔〕` `【】`

### 문장 부호

`※` `〃` `…` `‥` `∥` `—` `―` `′` `″`

### 원문자·로마자

한글 원문자 `㉠㉡㉢…`, 로마 숫자 대문자 `Ⅰ~Ⅹ`, 소문자 `ⅰ~ⅵ`

### 단위 기호

`㎡` `㎞` `㎝` `㎗` `㎏` `㎎` `℃` `㏊` `㏈` `㏗` 등 17종

### 기타 기호

`△▲▼` `→←↑↓↔` `㈜` `№` `§` `©` `®` `™`

## 기술 스택

- **단일 파일** — `index.html` 하나로 완결 (별도 서버·빌드 불필요)
- **Tailwind CSS** — CDN 방식으로 로드
- **Pretendard GOV** — 공공서체 적용
- **Material Icons** — 탭 및 UI 아이콘
- **Vanilla JS** — 프레임워크 없이 순수 JavaScript
- **localStorage** — 최근 복사 기록 영구 저장

## 데이터 구조

문자 데이터는 `[문자, 이름, 카테고리]` 형태의 배열로 관리됩니다.

```js
var D = [
  ['★', '검은별', 'pop'],
  ['→', '오른쪽화살표', 'arrow'],
  // ...
];
```

문서작성 탭은 별도의 `DOC_SECTIONS` 배열로 카테고리 블록 구조를 정의합니다.

```js
var DOC_SECTIONS = [
  {
    id: 'doc-seq',
    title: '항목 구분 기호',
    badge: '규정 순서',
    badgeClass: 'badge-rule',
    icon: 'format_list_numbered',
    seq: [ /* 항목 순서 시각화 데이터 */ ],
    chars: [ ['①','원숫자1'], /* ... */ ]
  },
  // ...
];
```
